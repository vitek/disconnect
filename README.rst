Disconnect art-project firmware
===============================

Device is based on atmega128 microcontroller.

Pinout
------
 * ring: PG1, PB6, PA3
 * hang: PD2
 * numeric (nc): PD3
 * speaker: PORTC, 8-bit
 * AT45DB642x is connected to SPI bus
 * transformer and ?flash power-down: unknown
 * UART
 * mode: unknown
